# ⭐ Interactive Rating Component

A simple interactive rating component where users can select a rating and submit their feedback dynamically.

## 📸 Preview
![preview](preview.jpg)

## 🚀 Features

- Select rating from 1 to 5  
- Active selected state  
- Dynamic UI update after submit  
- Responsive design  
- Smooth interactive experience  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript  

## 📌 Goal

Practice DOM manipulation, event handling, dynamic UI states, and interactive user experience.
